## Selected Projects

| Project | Overview | Key Result | Technologies |
|----------|----------|------------|--------------|
| **[PI_Detection](https://github.com/er-rahul-agarwal/PI_Detection)** | Diffusion-based framework for real-time pre-ignition anomaly detection in hydrogen engines | **AUROC = 1.00**, 100% detection across 8,000 cycles with **2.64 ms** early warning | PyTorch • Diffusion Models • Transformers |
| **Thermoacoustic Instability Predictor** | Deep autoencoder for predicting combustion instability using high-frequency pressure measurements | Processed **485M+ sensor readings**, predicting all instability events with **1.7–18.6 min** warning | Autoencoders • Signal Processing • Time-Series Analysis |
| **Flame Instability Monitor** | CNN-LSTM architecture for real-time combustion transition detection using flame chemiluminescence imaging | **98.7% accuracy** with **1.2 ms/frame** inference | CNN • LSTM • Computer Vision |

> **Note:** Several projects were developed through industrial collaborations and are subject to confidentiality agreements. While complete source code cannot be released, I am happy to discuss methodologies, system design, and implementation details.
